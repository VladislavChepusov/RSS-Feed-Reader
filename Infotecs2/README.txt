Добрый день.
Данный архив является решением для второго тестово задания(Rss фидер) компании infotecs по направлению "Разработчик С#"

Файл настроек(config.xml) хранит в себе ссылку на ленту,которую обрабатывает фидер (url) и частоту обновления ленты(update_time) 
представленное в милисекундах.

При запуске программы основное окно должно содержит список элементов ленты (название + дата).
При двойном нажатии на заголовой открывается браузер(или в уже открытом браузере должна открываться ссылка).
Описание статьи выводится при наведении мышки на элемент.Имеется возможность смотреть форматированное и неформатированное описание.

Имеется возможность изменить ленту и частоту обновления в меню настроек.Опции в меню настроек валидируются.
